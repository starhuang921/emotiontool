# 情感小工具 EmotionTool

一个基于 uni-app + Java Servlet 开发的情感类综合工具，包含情感运势、情绪指南、情书生成、情感小测试等功能，支持微信小程序、H5 等多端运行。

## 技术栈

- 前端：uni-app、Vue3、SCSS
- 后端：Java Servlet、Tomcat
- AI 接口：DeepSeek API
- 部署：本地调试 + 真机预览

## 功能模块

- 情感运势查看
- 情绪心理指南
- AI 情书一键生成
- 情感小测试
- 个人中心

## 项目结构

- emotiontool/
- ├── frontend/ # uni-app 前端项目
- └── serverend/ # Java 后端服务

## 运行方式

### 前端
使用 HBuilderX 打开 `frontend` 目录，直接运行到微信小程序或浏览器预览。

### 后端
使用 IDEA/Eclipse 打开 `serverend` 项目，配置 Tomcat 启动。
